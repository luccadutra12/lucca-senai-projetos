Lucca SENAI Projetos
 Repositório criado para armazenar todas as atividades, exercícios e projetos desenvolvidos durante minha formação no SENAI.

Sobre: 
 Este repositório reúne práticas de programação, lógica e desenvolvimento web, aplicando conceitos aprendidos em aula como:

Lógica de programação:
 JavaScript
 HTML e CSS
 Estruturas de repetição (for, while)
 Arrays e matrizes
 Manipulação de DOM
 Boas práticas de código (Clean Code)
 
 Estrutura:
 Os projetos estão organizados por atividades ou temas:
 atividades/

 script.js
(A estrutura pode evoluir conforme novos conteúdos forem desenvolvidos.)

Projetos:

Sistema de Boletim Escolar
 Mini jogos em JavaScript
 Páginas web interativas

Desenvolver habilidades em programação e construir uma base sólida para atuação na área de tecnologia.
Este repositório está em constante atualização conforme avanço no curso.
